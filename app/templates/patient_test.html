üí° Testing Instructions

Test 1: No Patient Selected

  1. Clear CCOW vault: 
    curl -X DELETE http://localhost:8001/ccow/active-patient \
  -H "Content-Type: application/json" \
  -d '{"cleared_by": "test"}'
  2. Click "Refresh Patient from CCOW" button above
  3. Should see: "No patient selected"

Test 2: Patient Selected

  1. Set a patient in CCOW: 
    curl -X PUT http://localhost:8001/ccow/active-patient \
  -H "Content-Type: application/json" \
  -d '{"patient_id": "ICN100001", "set_by": "test"}'
  2. Click "Refresh Patient from CCOW" button above
  3. Should see: Dooree, Adam (M, 45) with full demographics

Test 3: Different Patient

  1. Set a different patient: 
    curl -X PUT http://localhost:8001/ccow/active-patient \
  -H "Content-Type: application/json" \
  -d '{"patient_id": "ICN100002", "set_by": "test"}'
  2. Click "Refresh Patient from CCOW" button above
  3. Should see: Miifaa, Barry demographics

Test 4: VA V-Format ICN
